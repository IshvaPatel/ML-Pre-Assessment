Q1.

To determine the number of multiplications and additions needed to perform a matrix multiplication between a ((n, k)) matrix (A) and a ((k, m)) matrix (B), consider the following:

  • Each element in the resulting ((n, m)) matrix (C) is computed as the dot product of a row from (A) and a column from (B).
  • For each element in (C), this involves (k) multiplications and (k-1) additions.

Therefore, for each of the (n * m) elements in (C):

  • The total number of multiplications is (n * m * k).
  • The total number of additions is (n * m * (k - 1)).

#List of List Method
import time
from datetime import datetime

def matrix_multiply_list(A, B):
    n = len(A)
    k = len(A[0])
    m = len(B[0])

    C = [[0 for _ in range(m)] for _ in range(n)]

    for i in range(n):
        for j in range(m):
            for r in range(k):
                C[i][j] += A[i][r] * B[r][j]

    return C

# Method 1: Finding median with multiple passes (O(n^2))
def find_median_naive(arr):
    n = len(arr)
    mid = n // 2
    for i in range(mid + 1):
        max_idx = 0
        for j in range(1, n):
            if arr[j] > arr[max_idx]:
                max_idx = j
        if i == mid:
            return arr[max_idx]

# Method 2: Sorting the array and finding the median (O(n log n))
def find_median_sort(arr):
    arr_sorted = sorted(arr)
    n = len(arr_sorted)
    if n % 2 == 1:
        return arr_sorted[n // 2]
    else:
        return (arr_sorted[n // 2 - 1] + arr_sorted[n // 2]) / 2

arr = np.random.randint(0, 1000000, 10000)

Q12. Broadcasting is a powerful mechanism in NumPy that allows operations on arrays of different shapes. It avoids making unnecessary copies of data and allows vectorized operations, which makes computations faster and more memory efficient. Here's a small example to explain broadcasting: Suppose we have a 2D array (matrix) and we want to add a 1D array (vector) to each row of the matrix. Normally, we'd have to duplicate the vector to match the shape of the matrix. With broadcasting, NumPy automatically handles this for us.
